## 一、授受動詞概述

### 1. 授受動詞的作用
- 表達給予和接受的動作
- 用於描述人際關係中的互動
- 練習問題：
  ```
  1. 授受動詞の基本的な使い方
  2. 場面に応じた表現の選択
  3. 文脈の理解と応用
  ```

#### 例子：
- 日本語：私は彼に本をあげました。
- 英語：I gave him a book.
- 中文：我給了他一本書。
- 練習：以下の文を授受動詞を使って表現してください
  ```
  1. 我送給朋友一個禮物
  2. 老師給了我一本書
  3. 我從父母那裡收到錢
  ```

### 2. 使用授受動詞的場合
- 日常對話中表達給予和接受
- 描述人際互動和關係
- 練習問題：
  ```
  1. 日常会話での授受動詞
  2. ビジネスシーンでの授受動詞
  3. ニュースでの授受動詞
  ```

#### 例子：
- 日本語：彼女は私にプレゼントをくれました。
- 英語：She gave me a present.
- 中文：她給了我一個禮物。
- 練習：以下の状況で授受動詞を練習してください
  ```
  1. 友達との贈り物のやり取り
  2. 家族とのお金のやり取り
  3. 職場での資料の受け渡し
  ```

## 二、授受動詞的基本用法

### 1. 動詞的授受變化
- 「あげる」：表示給予
  - 例：私は彼に本をあげました。
- 「くれる」：表示給予（對方給我）
  - 例：彼女は私にプレゼントをくれました。
- 「もらう」：表示接受
  - 例：私は彼から本をもらいました。
- 練習問題：
  ```
  1. 動詞の変化規則の理解
  2. 各動詞の使い分け
  3. 実際の文での応用
  ```

#### 例子：
- 日本語：私は友達から手紙をもらいました。
- 英語：I received a letter from a friend.
- 中文：我從朋友那裡收到了一封信。
- 練習：以下の動詞を使って文を作ってください
  ```
  1. あげる：
  2. くれる：
  3. もらう：
  ```
